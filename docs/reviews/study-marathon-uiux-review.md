# UI/UX改善 実装解説

## 1. デザインコンセプト
**「Retro Nostalgic (レトロ・ノスタルジック)」**

学習の記録を「無機質なデータ」ではなく、「積み重ねてきた物語」として感じられるようなデザインを目指しました。
古いタイプライターや紙の質感を想起させるフォントと配色を採用し、デジタル疲れを軽減しつつ、落ち着いて学習に取り組める環境を提供します。

## 2. スタイル: Retro Nostalgic
今回のUIには、温かみと信頼感を演出する **Retro Nostalgic** テーマを採用しました。

### 採用理由
1.  **落ち着きと集中**:
    -   クリーム色の背景とアースカラーの配色は、目に優しく、長時間の学習記録の振り返りに適しています。
2.  **物語性**:
    -   セリフ体（Playfair Display）とモノスペースフォント（Courier Prime）の組み合わせにより、学習ログの一つ一つが「手記」のような重みを持ちます。
3.  **Bootstrap 5.3の活用**:
    -   標準的なUIフレームワークであるBootstrapを採用することで、保守性と拡張性を担保しつつ、カスタムCSS変数で独自の世界観を表現しました。

## 4. 実装のポイント (Bootstrap 5.3 + Custom CSS)

### 共通レイアウト (`layout/base.html`)
-   **Bootstrap 5.3**: グリッドシステムやコンポーネント（Navbar, Card, Alertなど）の基盤として採用。
-   **Custom Theme**: `src/main/resources/static/css/style.css` にCSS変数を定義し、Bootstrapのデフォルトスタイルをオーバーライド。
    -   `border-radius: 4px`: 角丸を小さくし、少し硬い印象を残すことでレトロ感を表現。
    -   `box-shadow`: ぼかしのない（solid）影を採用し、クラシックな印刷物のような立体感を演出。

### コンポーネント改修
1.  **Navbar**:
    -   Bootstrapの標準Navbarを使用しつつ、下線（`border-bottom`）と影を追加。ロゴにはバッジスタイルを適用。
2.  **Card UI**:
    -   ダッシュボードやログイン画面のカードは、`card` クラスをベースに、太めのボーダーと影を追加。
3.  **Typography**:
    -   見出しには `Playfair Display`、本文には `Inter`、データ表示には `Courier Prime` を使い分け、情報の階層構造を視覚的に明確化。
